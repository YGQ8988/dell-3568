# jmeter数据库连接
1.在测试计划界面最下面点击浏览(Add directory to jar to classpath右边的浏览），选择下载好的mysql-connector-java-x.x.x.jar包
2.添加咸成祖>JDBC Connection Configuration
3.配置JDBC Connection Configuration底部Database Connection Configuration
```
Database URL(固定格式):jdbc:数据库类型://IP:PORT/数据库名 例：jdbc:mysql://127.0.0.1:3306/blog_db
JDBC Driver class(固定格式):根据数据库类型选择 例：com.mysql.jdbc.Driver
Username：数据库连接用户名
Password：数据库连接用户名密码
```
